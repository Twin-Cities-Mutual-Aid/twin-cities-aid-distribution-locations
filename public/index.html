Search terms and examples:

Please note that this is not a complete list of search terms to use, but rather a general guide.

  • Zip Code - e.g. '55108'
  • City Name - e.g. 'St. Paul'
  • Item - e.g. 'food', 'water', or 'toiletries'
  • Building Type - e.g. 'church', or 'shelter'
  • Language Criteria - e.g. 'bilingual', or 'spanish'
  • Site Need - e.g. 'needs volunteers', or 'seeking donations'
